# Movies Data Analysis Dashboard 🎬

## 📌 Objective
The objective of this project is to analyze movies dataset to identify trends in movie releases, awards, nominations, and duration.

## 📊 Description
This dashboard provides insights into:
- Total Movies
- Awards & Nominations
- Average Duration
- Genre-wise and Year-wise analysis

The dashboard is interactive and allows filtering by release year and genre.

## 🛠️ Tools Used
- Power BI
- Excel / CSV
- Data Visualization

## 🔍 Key Insights
- Consistent movie production across years
- Genre-based award distribution
- Duration trends over time
